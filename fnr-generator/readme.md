## Install

```
$ npm install --save fnr-generator
```

## Usage

```javascript test
import generator from 'fnr-generator'

var gen = generator(new Date('2015-03-24'))

gen.next()
//=> { done: false, value: '24031550005' }

gen.next()
//=> { done: false, value: '24031550196' }
```
